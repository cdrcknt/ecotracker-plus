## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/cdrcknt/ecotracker-plus.git
cd eco-tracker-plus
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
